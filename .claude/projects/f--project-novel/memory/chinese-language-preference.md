本项目要求所有交流使用简体中文：

1. **聊天回复** — 与用户对话全部使用简体中文
2. **思考过程** — `<thinking>` 标签内的推理用简体中文
3. **代码注释** — 所有形式的注释使用简体中文
4. **Git 提交信息** — 使用简体中文
5. **文档** — Markdown、README 等使用简体中文

**例外情况（允许英文）：**
- 代码标识符（变量名、函数名、类名）
- 通用技术术语（API、HTTP、JSON、SQL）
- 配置键值、环境变量名
- 文件路径、URL
- 命令行指令

**Why:** 用户明确要求整个项目使用简体中文，以确保代码和文档对中文团队的可读性。

**How to apply:** 在所有回复、思考、注释、提交、文档中默认使用简体中文，仅代码标识符、标准技术术语、配置项等保留英文。
